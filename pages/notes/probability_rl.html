The Cramer-Chernoff Method and Subgaussian Random Variables

(Subgaussianity): A random variable X is σ-subgaussian if for all λ∈R, it holds that E[exp⁡(λX)]≤exp⁡(λ2σ2/2).

Moment-generating function: MX:R→R defined by MX(λ)=E[exp⁡(λX)].

Cumulant-generating function: ψX(λ)=log⁡MX(λ)≤12λ2σ2 for all λ∈R.

THM: If X is σ-subgaussian, then for any ϵ≥0, P(X≥ϵ)≤exp⁡(−ϵ22σ2).

Proof (Cramer-Chernoff Method): P(X≥ϵ)=P(exp⁡(λX)≥exp⁡(λϵ))≤E[exp⁡(λX)]exp⁡(−λϵ) (Markov’s Ineq.)≤exp⁡(λ2ϵ22−λϵ). (Def. of subgaussianity) Choosing λ=ϵ/σ2 completes the proof. ◻

Therefore, P(X≥2σ2log⁡(1/δ))≤δ, hence X takes values in the interval (−2σ2log⁡(2/δ),2σ2log⁡(2/δ)).

Properties: Suppose that X is σ-subgaussian and X1, X2 are independent and σ1 and σ2-subgaussian, then:

  • E[X]=0 and V[X]≤σ2.
  • cX is |c|σ-subgaussian for all c∈R.
  • X1+X2 is X12+X22-subgaussian.
